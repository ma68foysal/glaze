# Glaze

A single-product Shopify theme for beauty and skincare brands. Built on Shopify's [Skeleton theme](https://github.com/Shopify/skeleton-theme) base.

## Local development

Requires the [Shopify CLI](https://shopify.dev/docs/api/shopify-cli).

```bash
shopify theme dev
```
